## 1. 문제 정의

### 원인 분석

**3가지 주요 누수 경로 발견:**

1. **keepAliveInterval 누수** (main.js:158)
   - Interval ID 미저장 → 앱 종료 시 미정리
   - 250ms마다 실행되어 지속적 메모리 점유

2. **Map 리소스 누수** (main.js:340-342, 617)
   - `firstPreToolUseDone`, `postToolIdleTimers`, `sessionPids` Map
   - 에이전트 종료 시 일부만 정리되어 누락 케이스 존재

3. **Renderer interval 누수** (renderer.js:35)
   - `agentStates` Map의 interval 참조
   - 에이전트 제거 시 interval이 정리되지 않음

### 영향도

- **사용자:** 4시간 사용 시 앱 충돌로 이어짐
- **시스템:** 불필요한 메모리 점유로 다른 앱 성능 저하
- **비즈니스:** 파워 유저(핵심 사용자층) 이탈 위험
